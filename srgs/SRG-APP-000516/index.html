SRG-APP-000516 Controls

STIG ID Version Title Product
SRG-APP-000516-MFP-000195 V3R3 The Mainframe Product must be configured in accordance with the security configuration settings based on DoD security configuration or implementation guidance, including STIGs, NSA configuration guides, CTOs, and DTMs.
SRG-APP-000354-MFP-000136 V3R3 The Mainframe Product must provide the capability for authorized users to select a user session to capture/record or view/hear.
SRG-APP-000355-MFP-000139 V3R3 The Mainframe Product must provide the capability for authorized users to remotely view/hear, in real time, all content related to an established user session from a component separate from the Mainframe Product being monitored.
