SRG-NET-000019 Controls

STIG ID Version Title Product
SRG-NET-000019-FW-000003 V3R1 The firewall must be configured to use filters that use packet headers and packet attributes, including source and destination IP addresses and ports, to prevent the flow of unauthorized or suspicious traffic between interconnected networks with different security policies (including perimeter firewalls and server VLANs).
SRG-NET-000019-FW-000004 V3R1 The firewall must immediately use updates made to policy enforcement mechanisms such as firewall rules, security policies, and security zones.
