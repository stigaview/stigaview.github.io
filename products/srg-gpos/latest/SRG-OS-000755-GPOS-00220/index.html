The operating system must monitor the use of maintenance tools that execute with increased privilege.

STIG ID: SRG-OS-000755-GPOS-00220  |  SRG: SRG-OS-000755 | Severity: medium |  CCI: CCI-004188

Vulnerability Discussion

Maintenance tools that execute with increased system privilege can result in unauthorized access to organizational information and assets that would otherwise be inaccessible.

Check

Verify the operating system is configured to monitor the use of maintenance tools that execute with increased privilege.

If the operating system is not configured to monitor the use of maintenance tools that execute with increased privilege, this is a finding.

Fix

Configure the operating system to monitor the use of maintenance tools that execute with increased privilege.
