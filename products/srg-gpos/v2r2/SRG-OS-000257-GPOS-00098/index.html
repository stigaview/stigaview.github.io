The operating system must protect audit tools from unauthorized modification.

STIG ID: SRG-OS-000257-GPOS-00098  |  CCI: CCI-001494 |  SRG: SRG-OS-000257 | Severity: medium

Vulnerability Discussion

Protecting audit information also includes identifying and protecting the tools used to view and manipulate log data. Therefore, protecting audit tools is necessary to prevent unauthorized operation on audit information.

Operating systems providing tools to interface with audit information will leverage user permissions and roles identifying the user accessing the tools and the corresponding rights the user has in order to make access decisions regarding the modification of audit tools.

Audit tools include, but are not limited to, vendor-provided and open source audit tools needed to successfully view and manipulate audit information system activity and records. Audit tools include custom queries and report generators.

Check

Verify the operating system protects audit tools from unauthorized modification. If it does not, this is a finding.

Fix

Configure the operating system to protect audit tools from unauthorized modification.
